# Design Notes

## Intended Feel

The website should communicate:
- technical depth
- calm precision
- strong research identity
- modern editorial taste

It should not read like a generic startup portfolio or a crowded academic page.

## Visual Principles

- Use a restrained palette with one strong accent.
- Favor clear typography and spacing over decorative noise.
- Let sections breathe.
- Use contrast intentionally.
- Keep motion subtle and meaningful.

## Layout Principles

- Start with a strong hero and concise value statement.
- Keep section order simple and scannable.
- Use modular sections that can evolve independently.
- Design for mobile first, then expand.

## Content Principles

- Prefer short, high-signal writing.
- Explain work in plain language before technical detail.
- Avoid CV-style information overload on the homepage.
- Link out for deeper material such as papers, GitHub, Scholar, and CV.
